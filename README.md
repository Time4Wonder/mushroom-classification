# Mushroom-Classification

## Übersicht

Dieses Projekt beschäftigt sich mit einem **Multi-Klassifikationsproblem** auf Basis eines Pilz-Datensatzes. Das Ziel ist es, verschiedene Merkmale eines Pilzes zu analysieren und seine **Essbarkeit** vorherzusagen. 

Das Projekt dient der Vertiefung meiner **Machine Learning**-Kenntnisse und dem praktischen Umgang mit Datenanalyse und Modellentwicklung.

## Datensatz

Der Datensatz stammt von der UCI Machine Learning Repository:
- **Quelle**: [Mushroom Dataset](https://archive.ics.uci.edu/dataset/73/mushroom)
- **Klassifizierungsziel**: Essbarkeit von Pilzen (binäre Klassifikation)
- **Merkmale**: Verschiedene physikalische und biologische Charakteristiken

## Modelle

In diesem Projekt werden verschiedene Machine Learning-Modelle trainiert und evaluiert, um die beste Vorhersagegenauigkeit zu erreichen.
